=== Spring Boot Interview Questions ===

Q1: What is Spring Boot?
A1: Spring Boot is a framework that simplifies Spring application development with auto-configuration.

Q2: What does @SpringBootApplication do?
A2: It combines @Configuration, @EnableAutoConfiguration, @ComponentScan.

Q3: What is Spring Boot Starter?
A3: Dependency descriptor that bundles common dependencies for specific features.
What is Spring Boot?
Answer: Spring Boot is a framework built on top of Spring that simplifies application development by providing auto-configuration, embedded servers (Tomcat/Jetty), and starter dependencies.

MNC-level insight: It enables rapid development of production-ready applications with minimal boilerplate.
